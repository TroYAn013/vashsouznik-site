VashSouznik — лендинг первой партии пантерного мухомора (Иркутск)

Публикация на GitHub Pages:
1) Создайте публичный репозиторий (например, vashsouznik-site).
2) Загрузите ВСЕ файлы из архива в корень репозитория.
3) Settings → Pages → Branch: main → Folder: /(root) → Save.
4) Сайт будет доступен по адресу: https://<ваш_логин>.github.io/<имя_репозитория>/

Где заменить:
- В index.html обновите @VashSouznik при необходимости.
- Положите свои реальные фото в папку /img/ с именами photo1.jpg..photo4.jpg (уже есть базовые изображения).
